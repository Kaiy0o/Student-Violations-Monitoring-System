# Get started

### 1. Clone the Repository

```bash
git clone https://github.com/Kaiy0o/Student-Violations-Monitoring-System
```

### 2. Get Dependencies

```bash
flutter pub get
```

### 3. Build instruction

```bash
flutter build apk
flutter run
```
